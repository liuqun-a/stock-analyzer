### ETF 监控（561570, 159206）
- **频率**：每 5 分钟一次
- **时段**：交易日 9:00-11:30, 13:00-15:30
- **预警**：涨跌幅 ±1%

### 个股监控（002416）
- **频率**：每天 10:00 和 14:00 各一次
- **预警条件**（双重监控）：
  1. 实时价格 < ¥12.00 → 预警
  2. 当天最低价 < ¥12.00 → 预警
- **推送**：飞书实时通知

### 推送方式
- **飞书实时推送**
- 相同预警 5 分钟内不重复

## 🔄 添加新股票

编辑 `config/stocks.yaml`：
```yaml
custom_watch:
  - code: "新代码"
    name: "股票名称"
    watch_price: 基准价
    alert_pct: 1  # 预警阈值
    enabled: true
```

编辑 `config/alerts.yaml`：
```yaml
custom_thresholds:
  "新代码":
    alert_pct: 1
    base_price: 基准价
```
